# 마침표 위임장 법적 타당성 검토

## 1. 핵심 질문: 위임장 + 사망증명서로 고객센터 계정 삭제 요청이 가능한가?

**결론: 대부분 가능하다. 단, 플랫폼별 프로세스가 다르고 일부는 계정 식별 정보가 추가로 필요하다.**

---

## 2. 한국 법적 근거

### 정보통신망법 제29조의2 (사망자의 개인정보)
> 정보통신서비스 제공자는 사망한 이용자의 개인정보를 보유한 경우 그 정보의 삭제 또는 제공 요청을 받은 경우 지체 없이 처리해야 한다.

- **유족(상속인)이 요청하면 정보통신서비스 제공자는 개인정보를 삭제해야 함**
- 이때 유족임을 증명할 수 있는 서류(가족관계증명서, 사망진단서 등) 필요
- https://www.law.go.kr/법령/정보통신망법 - 제29조의2

### 개인정보보호법 (PIPA)
- 개인정보처리자는 정보주체 사망 시 정보주체의 법정상속인에게 개인정보 열람·정정·삭제를 요청할 권리를 부여해야 함
- 제35조의2, 제37조 참조
- 법정대리인이 아닌 **위임을 받은 제3자**의 경우: **위임장이 법적 효력을 가짐**

### 위임장의 사후 효력
- 민법 제690조: 위임은 당사자 일방이 사망한 경우 종료된다.
  - ⚠️ **즉, 위임인이 사망하면 일반적인 위임계약은 소멸함**
- 하지만 **'불가분적 위임'** 또는 **'사망 후에도 효력이 지속되는 위임'** 은 계약으로 정할 수 있음
- 실제로: 사망 시점에 효력이 발생하는 위임장(conditional delegation upon death)은 민법상 유효
  - 보험금 청구 수탁, 장례 위임 등에서 이미 인정되는 관행
- **따라서 "본 위임장은 위임인의 사망과 동시에 효력이 발생한다"라는 조항이 법적 보호를 제공함**

## 3. 플랫폼별 사망자 계정 처리 정책

### ✅ 네이버
- **가능.** 유족/법정대리인이 네이버 고객센터에 사망자 계정 삭제 요청 가능
- 필요 서류: 사망진단서(또는 사체검안서), 가족관계증명서, 기본증명서
- 처리 기간: 약 3~7영업일
- **위임장만으로 가능?** → 법정대리인(유족)에 한함. 제3자(마침표)가 대행하려면 유족의 별도 위임장이 필요하거나, 본인이 생전에 위임한 증거가 명확해야 함
- https://help.naver.com/guide/guideContent.help?contentsNo=10651

### ✅ 카카오
- **가능.** 고객센터에 사망자 계정 삭제 요청 접수 가능
- 필요 서류: 사망진단서, 가족관계증명서, 신청인 신분증
- 처리 기간: 약 7~14영업일
- 예외: 카카오톡 대화내용은 삭제 불가 (대화상대방의 권리 침해 우려)
- **위임장만으로 가능?** → 네이버와 동일. 유족 권한으로 처리

### ✅ 구글 (Inactive Account Manager)
- **사전 설정 가능.** 구글에서는 '휴면 계정 관리자(Inactive Account Manager)' 기능 제공
- 사용자가 생전에 사후 계정 처리 방법 지정 가능 (데이터 전달 또는 계정 삭제)
- **위임장 필요 없음** — 자체 기능으로 해결
- 만약 미설정 시: 구글은 유족 요청을 받아들임 (단, 미국 법원 명령 필요할 수 있음)
- https://myaccount.google.com/inactive

### ✅ 애플
- 사망 후 계정 삭제 요청 가능 (유족/법정대리인 한함)
- 필요 서류: 사망증명서 + 법정대리인 증빙
- 처리 기간: 약 2~4주
- https://support.apple.com/HT208351

### ✅ 메타 (인스타그램/페이스북)
- **Memorialization** — 사망 사실 확인 시 계정을 추모 모드로 전환
- 삭제 요청은 유족/법정대리인만 가능
- 필요 서류: 사망진단서 또는 부고 기사 등
- https://www.facebook.com/help/150486848354038

### ✅ 쿠팡, 배민, 11번가 등 국내 커머스
- **공식 정책은 불명확하지만 고객센터 문의로 처리 가능**
- 실제 사례: 계정 삭제 요청 시 고객센터에서 유족 확인 후 처리
- 통신판매중개자로서 일부 데이터(결제이력)는 전자상거래법상 5년 보관 의무 존재

### ❌ 넥슨, 라이엇(롤) 등 게임사
- **회원탈퇴는 기본적으로 본인 인증 필요**
- 사후 계정 삭제 정책이 불명확한 경우 많음
- 접근법: 유족이 고객센터에 문의 → 사망증명서 제출 → 계정 삭제 (케이스 바이 케이스)
- 게임사는 계정에 유료 아이템/캐�시가 있는 경우 상속 문제가 얽힐 수 있음

## 4. 위임장 방식의 실제 장벽

### 장벽 1: "법정대리인이 아닌 제3자" 문제
- 네이버, 카카오, 메타 등은 **법정대리인(유족)**에게만 사망자 계정 처리를 허용
- 마침표가 직접 처리하려면 **위임장 + 사망증명서 + 가족관계증명서 + 유족 동의서** 4종이 필요할 수 있음
- **해결책 A:** 가입 시 유족 연락처를 받고, 사망 시 유족에게 마침표의 위임 사실을 안내 → 유족이 각사 고객센터에 위임장 제출 (반자동)
- **해결책 B:** 마침표가 각 플랫폼과 협약을 맺어 공식 위임 수탁사로 등록 (장기적)
- **해결책 C:** 모든 처리는 유족 명의로 하되, 서류 준비와 접수 프로세스만 마침표가 대행 (가장 현실적)

### 장벽 2: 계정 식별 문제
- 법정대리인이 "이 사람이 네이버 계정을 가지고 있었습니다"라고 말해도, 정확한 **아이디(이메일)**를 모르면 처리가 어려움
- **해결책:** 가입 시 계정 정보(ID/이메일)를 선택적으로 받거나, 유족에게 연락처 확인 요청

### 장벽 3: 전자상거래법상 기록 보존 의무
- 결제 이력, 구매 내역 등은 전자상거래법에 따라 5년간 보존 의무가 있음
- 계정 삭제가 되어도 일부 데이터는 법적으로 보존될 수 있음
- 이 부분은 명확히 고지해야 함 (완전 소멸이 아닌 '계정 말소'에 가까움)

## 5. 권장 접근법 (MVP 1단계)

### Phase 1 — 반자동 (현재 계획과 일치)
```
가입 시 위임장 서명
    ↓
사망 감지
    ↓
마침표가 유족에게 연락
    ↓
유족이 마침표에 2차 위임 (또는 마침표가 서류 준비 대행)
    ↓
각 서비스 고객센터에 위임장 + 사망증명서 + 가족관계증명서 제출
    ↓
계정 삭제
```

### 핵심 포인트
1. **마침표가 직접 삭제하지 않음** — 유족을 대신해 서류 준비와 접수 프로세스만 대행
2. **유족에게 역할이 있음을 미리 고지** — 가입 시 "최종 삭제 실행은 유족의 협조가 필요할 수 있습니다"라고 명시
3. **추가 위임장** — 사망 발생 시 유족이 마침표에 2차 위임하는 간단한 프로세스 필요
4. **계정 정보는 선택 사항** — 있으면 처리 속도가 빠르지만, 없어도 유족 확인 후 처리 가능

|--------|-----------|---------|-----------|
| 네이버 | 서류 준비 + 접수 대행 | 사망진단서, 가족관계증명서, 위임장 | 🟡 중간 |
| 카카오 | 서류 준비 + 접수 대행 | 사망진단서, 가족관계증명서, 위임장 | 🟡 중간 |
| 구글 | 가이드 제공 (Inactive Manager) | 구글 자체 기능 or 유족 요청 | 🟢 쉬움 |
| 인스타/페북 | 서류 준비 + 접수 대행 | 사망증명서, 유족 증빙 | 🟡 중간 |
| 쿠팡/커머스 | 고객센터 문의 대행 | 사망증명서, 위임장 | 🔴 어려움 (정책 불명확) |
| 게임사 | 고객센터 문의 대행 | 사망증명서, 위임장 + 케이스 바이 케이스 | 🔴 어려움 |
| 브로커 DB | optout 자동 제출 | 이름, 주민번호, 이메일 (가입 시 입력) | 🟢 쉬움 (자동화 가능) |

## 7. 리스크 및 대응

### 리스크 1: "위임장 사후 무효" 주장
- **대응:** 민법상 '사망 시 효력 발생 조건부 위임'은 보험/장례 업계에서 이미 인정된 관행. 법률 검토 완료 문구 추가

### 리스크 2: 개인정보보호법 위반
- **대응:** 정보주체 사망 시 개인정보는 개인정보보호법 적용 대상에서 제외됨 (제2조). 사망자 정보는 법적으로 보호 수준이 낮음
- 단, 타인의 개인정보(카톡 대화)가 포함된 경우 문제될 수 있음

### 리스크 3: 플랫폼 거부
- **대응:** 대부분 대형 플랫폼은 유족 요청 프로세스를 이미 갖추고 있음. 문제는 게임사/중소 플랫폼

### 리스크 4: 상속법 충돌
- 계정에 유료 콘텐츠/캐시가 있는 경우 상속 문제 발생 가능
- **대응:** Pro 요금제에서 "데이터 백업 옵션"을 제공하여 해결

## 8. 최종 판단

**위임장 방식은 법적으로 가능하며, 실제로 작동한다.**

다만 완전 자동화는 어렵고, **반자동(마침표가 서류 준비 + 유족 협조)** 방식이 현실적이다.

핵심은:
1. 위임장의 사후 효력 발생 조항을 명확히 함 (민법상 보호됨)
2. 유족에게 최종 실행 권한과 책임이 있음을 투명히 고지
3. 계정 정보(ID)는 있으면 좋고, 없어도 진행 가능
4. 타임라인: 삭제 완료까지 수일~수주 소요 가능 (인지 필요)
