## 1. 이 서비스의 정체: 구독? 보험? 위임?

| 구분 | 전형적 예 | 마침표 |
|-----|----------|-------|
| 넷플릭스 | 콘텐츠 소비 | ❌ |
| 보험 | 사고 시 보상금 | ❌ |
| 상조 | 장례 서비스 | ❌ |

마침표는 **"사망 시 실행되는 위임 서비스"**. 보험처럼 느껴지지만 보험은 아니다 (보상금 없음). 구독처럼 느껴지지만 실시간 혜택은 없다. 이 애매한 정체성이 가격 책정을 어렵게 만든다.

## 3. 등기우편 전환 후 가치 변화

### Before: 디지털 서비스
```
"앱에서 자동으로 삭제됩니다"
→ 가치 인식: 낮음. "그냥 버튼 누르는 건데?"
→ 적정 가격: ₩1,000~₩3,000/월
```

### After: 등기우편 서비스
```
"위임장 + 사망증명서를 각 서비스에 등기 발송합니다"
→ 가치 인식: 높음. "아 진짜 사람이 하는구나, 서류가 오고 가는구나"
→ 적정 가격: ₩5,000~₩10,000/월
```

**등기우편으로 전환하면서 오히려 가격 인상이 가능해졌다.** 물리적인 증거(등기 영수증, 도장, 서류)가 있는 서비스는 디지털보다 프리미엄을 받을 수 있다.

## 4. 심리적 가격 분석

### 한국 소비자의 구독 지불 의향

| 금액대 | 인식 | 예시 |
|-------|-----|------|
| ~₩3,000 | "부담 없다, 한번 써보자" | BASIC 현재 |
| ₩5,000~₩7,000 | "괜찮은 가격이다" | 넷플릭스 1인 (₩5,500→₩6,500) |
| ₩10,000~₩15,000 | "프리미엄이다" | Spotify (₩10,900), 유튜브 프리미엄 (₩14,900) |
| ₩20,000+ | "고민 필요" | 워크플레이스/전문 서비스 |

### 가격 라운딩 효과
- ₩2,900 → ₩3,000? No. ₩2,900이 훨씬 싸 보임 (2,000원대 인식)
- ₩4,900 → ₩5,000? No. ₩4,900이 4,000원대 인식
- ₩7,900 → ₩8,000? 유의미한 차이 있음
- ₩9,900 → ₩10,000? 큰 차이 (1만원 vs 9,900원)

### 기준점 효과 (Anchoring)
- BASIC ₩2,900 = 하루 96원 = "커피 값도 안 됨"
- PRO = BASIC의 1.7배. 차이가 별로 안 나서 PRO 선택 유도 약함
- **현재 BASIC-PRO 차이가 너무 작다 (₩2,000 차이)**

## 5. 요금제 간 차별화 포인트 재검토

### 현재 문제: BASIC과 PRO의 차이가 모호함

```
BASIC  ₩2,900  — 계정 10개, 브로커, 유족 통지
PRO    ₩4,900  — 무제한, 게임 포함, 등기우편, 24시간
FAMILY ₩9,900  — 4인, Pro 전부
```

**문제 1:** 등기우편이 PRO에만 포함되어 있으면 BASIC은 디지털로 처리?  
→ 방금 논의한 것처럼 모든 티어가 등기우편 기반이어야 함

**문제 2:** BASIC-PRO 차이가 고작 ₩2,000  
→ 차이가 너무 작아 PRO가 "별로 안 프리미엄"처럼 보임

**문제 3:** FAMILY는 4인 기준 1인당 ₩2,475  
→ 지나치게 저렴. 가족 단위 가입 유인은 좋지만 수익성 낮음

### 권장: 티어 재구성

| 요금제 | 가격 (변경) | 핵심 차별점 |
|-------|-----------|-----------|
| BASIC | ₩2,900 | **→ 유지.** 등기우편 포함. 계정 10개. 브로커. |
| PRO | **₩6,900** | **→ 인상.** 계정 무제한. 게임 말소. 해외 EMS. 정보조회(이메일/전화로 계정 찾기). 속성 처리. |
| FAMILY | **₩12,900** | **→ 인상.** 4인. Pro 전부 + 가족 계정 동기화. |

### 변경 이유

**BASIC ₩2,900 (유지)**
- 하루 96원 = 진입장벽 최소화
- "일단 한번 써보자" 유도
- 마케팅 포인트: "하루 커피 값으로 내 디지털 흔적을 지키세요"
- MAU 1만 명 = 월 ₩29,000,000

**PRO ₩4,900 → ₩6,900 (42% 인상)**
- 등기우편 물리 서비스의 가치 반영
- BASIC과의 차이 ₩4,000 = "확실히 더 좋은 서비스" 인식
- 해외 EMS, 계정 정보 조회 등 실제 추가 비용 발생
- 마케팅 포인트: "국내외 모든 계정, 등기우편으로 말소"
- MAU 1만 명 중 30% = 월 ₩20,700,000

**FAMILY ₩9,900 → ₩12,900 (30% 인상)**
- 4인 기준 1인당 ₩3,225 = BASIC보다 약간 비쌈
- BASIC 4명 개별가입 (₩2,900×4=₩11,600)보다 비싸지만, 계정 공유/동기화 혜택
- 가족 단위 계약 유도 (churn rate 낮음)
- MAU 1만 명 중 10% = 월 ₩12,900,000

## 7. 최종 권장

**시나리오 B: ₩2,900 / ₩6,900 / ₩12,900**

### 이유

1. **BASIC ₩2,900** — 하루 96원. 이 가격이 **마케팅 무기**. "두 번 생각할 필요 없는 가격"

2. **PRO ₩6,900** — 등기우편 물리 서비스의 진정한 가치. BASIC과 차이를 확실히 벌려서 "PRO 들어야 제대로 된 서비스" 인식 형성. 계정 무제한 + 해외 EMS + 속성 = ₩6,900 납득 가능

3. **FAMILY ₩12,900** — 1인당 ₩3,225. 개별 BASIC(₩2,900)보다 약간 비싸지만, 계정 공유와 관리 편의성이 차별점

### 등기비 부담 없음 (재확인)

| 요금제 | 가입 1년 수익 | 등기비 | 비율 |
|-------|------------|-------|-----|
| BASIC | ₩34,800 | ₩75,000~₩100,000 | 215~287% |
| PRO | ₩82,800 | ₩90,000~₩120,000 | 109~145% |
| FAMILY | ₩154,800 | ₩300,000~₩400,000 (4인) | 194~258% |

**1년 이내 사망 시 적자 발생하지만,**
- 통계적으로 40세 가입자가 1년 내 사망할 확률: 0.05% 미만
- 가입자 10,000명 중 연간 약 30명 사망 = 대부분 장기 가입자
- 장기가입자의 누적 수익이 단기 적자를 충분히 상쇄

### 결론

> **등기우편 전환은 가격 인상의 기회다.**
> **"하루 96원" 진입장벽은 유지하되, PRO를 확실히 프리미엄화하라.**
> **물리적 서비스는 디지털보다 프리미엄을 받을 자격이 있다.**
